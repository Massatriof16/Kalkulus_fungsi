Aplikasi Kriptografi Web

TENTANG PROJECT


Sistem enkripsi pada proyek ini menggunakan konsep Affine Cipher, yaitu salah satu algoritma kriptografi klasik yang berbasis pada operasi aritmatika modular. Algoritma ini bekerja dengan cara mengubah setiap byte atau karakter menjadi bentuk terenkripsi menggunakan persamaan matematika berikut:

Rumus Enkripsi:
E(x) = (a × x + b) mod n

Rumus Dekripsi:
D(x) = a⁻¹ (x − b) mod n

Keterangan:
• x = nilai byte atau karakter asli
• a = kunci tetap (konstanta pengali)
• b = kunci geser (diambil dari PIN pengguna)
• n = modulo (menggunakan 256 untuk sistem byte)
• a⁻¹ = invers modular dari a terhadap n

Proses dekripsi hanya dapat dilakukan jika nilai a memiliki invers modular terhadap n, sehingga proses enkripsi dan dekripsi dapat berjalan secara benar dan konsisten.

CARA PENGGUNAAN


Encrypt & Decrypt Teks / file :
1. Atur Pin dan ingat pin sebagai encrypt atau decrypt (akan berlaku disemua fitur)
2. Masukkan teks biasa atau upload file Ke tabel encrypt atau pun decrypt
3. Tekan Tombol Encrypt atau Decrypt untuk menghasilkan output
4. Pastikan Pin untuk encrypt dan decrypt adalah sama sehingga bisa meghasilkan encrypt dan decrypt yang tepat
5. Hasil Output teks akan keluar di bawah dan dapat disalin
6. Output File akan muncul tempat downlod di bawah tombol

Kirim pesan Teks atau File melalui Telegram :
1. buka telegram dan gunakan pencarian, cari "Rose" jika ditemukan buka Lalu tekan tombol start
2. kirim pesan /id atau /id @username, Rose akan memberi tahu id mu atau id dari username yang kamu berikan
3. cari @encodec_bot setelah ditemukan buka dan tekan tombol start
4. Kembali ke web Masukkan Id target tujuan yang diberikan oleh Rose
5. Masukkan teks atau upload file untuk melakukan kiriman teks atau file hasil encrypt ke tujuan id anda
